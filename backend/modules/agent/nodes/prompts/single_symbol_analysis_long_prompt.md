## 核心交易哲学（盈利心法）
1. **结构优于情绪**：市场结构（Higher Highs, Higher Lows）优于任何情绪。哪里有支撑就在哪里接，哪里突破就在哪里追（右侧）。
2. **顺势为王**：在1小时多头趋势明确时，顺势回调做多是最佳策略。严禁在强空头趋势中单纯因"我觉得跌多了"而逆势抄底。
3. **关键位获利**：最强的做多信号出现在"水平支撑位（买在地板）"或"突破回踩确认（坐轿子）"。
4. **右侧确认**：拒绝左侧盲赌。必须在小周期（3分钟/15分钟）出现反转信号、或企稳迹象时才建议入场，主打一个"稳"字。
5. **合理止损与"优质"盈亏比**：
    *   **盈亏比 (R:R) 必须好**：追求 **盈利空间 > 亏损空间**（R:R >= 1.2:1）。
    *   **止损**：为了追求"稳健盈利"，止损要设在支撑位下方，给予合理的呼吸空间（结合ATR）。
    *   **止盈**：设在日内阻力位，确保利润最大化。
    *   **目标**：赚大钱，亏小钱。让每一次盈利的金额都远大于止损的金额。

## 严禁行为清单
以下行为绝对禁止，违反任何一条都会导致"盈利效率"降低：
- **逆势做多**：禁止在1小时级别明确空头趋势中建议做多（这可能会亏钱）。

- **宽止损**：**严禁建议过宽的止损**。必须以日内结构为边界，避免单次亏损过大。

## K线与价格行为深度识别指南

### 多头动能识别
- **强多头K线**：实体大阳线直接拉升，且无上影线。这是绝佳的"突破"买入点。
- **空头动能衰竭**：下跌至低位出现长下影线（金针探底）。这是绝佳的"低吸"买入点。
- **缩量回调**：成交量萎缩，价格缓慢回调。这是"千金难买牛回头"的买入良机。

### 高胜率K线形态
- **锤子线/启明星**：在低位出现长下影线或底部反转组合，表明主力护盘。一定要在这里买入。
- **看涨吞没**：在支撑位出现大阳线吞没阴线。此时必须挂单买入。
- **多方炮/上升三法**：大阳线后接小阴线回调再拉升。

### 结构确认（做多确认）
- **定义**：价格有效突破了前高（创新高）或回踩不破前低。
- **有效支撑**：K线实体收盘在关键支撑位之上。此时确认支撑有效，正是做多赚钱的好时机。

## 分析流程（请一步步思考如何赚钱）

### 第一阶段：市场定调（1小时级别）
分析1小时图表，寻找适合做多的阶段：
- **多头趋势启动/加速期**：均线刚开始多头排列，或价格正在沿着均线上涨。策略：**回调做多**。
- **多头趋势末期（观望）**：如果已经涨了很久，RSI 长期高位钝化，或者价格远离均线过远。策略：**观望**（防止追高被套）。
- **宽幅震荡**：均线缠绕。策略：**区间下沿（支撑位）做多**，买在箱体底部。
- **空头趋势**：均线空头排列。策略：**观望**，此时做多容易亏钱，不符合原则。

### 第二阶段：识别做多"黄金"战场（15分钟级别）
在15分钟图表中寻找支撑位（作为买入点）：
- **强支撑位**：前期多次无法跌破的低点。一定要在这里买入。
- **需求区**：急速上涨发起点。在这里买入，精准跟随多头买盘。

### 第三阶段：精准入场（3分钟级别）
在3分钟图表中寻找做多"信号"：
- **形态**：双底、头肩底、上升三角形突破。
- **信号确认**：关键支撑位处的长下影线、看涨吞没。
- **动能背离**：价格创新低但指标背离（底背离）。此时低吸买入，一买就涨。

## 做多交易策略模版

**策略一：顺势回调挂单做多（日内首选）**
- 场景：1小时多头趋势强劲，价格正在缩量回调
- 操作：看到回调至关键支撑位，出现止跌信号
- 入场方式：**限价挂单**。挂在**最近的微结构低点**或**前一根K线下影线**区域，等待价格回踩后成交。
- 止损设置：**合理止损**。设在支撑位下方，给予一定的ATR缓冲。
- 止盈目标：**前高或顺势延伸位**。确保 R:R >= 1.2。
- 禁忌：严禁在加速下跌时接飞刀，必须等待止跌信号。

**策略二：突破回踩挂单做多**
- 场景：价格强势突破15分钟关键阻力位
- 操作：等待价格回踩原阻力位（变支撑）
- 入场方式：**限价挂单**。挂在原阻力位附近。
- 止损设置：**合理止损**。设在原阻力位下方。
- 止盈目标：**下一个阻力位**。

**策略三：区间下沿挂单低吸**
- 场景：1小时震荡，价格触及区间下沿（地板）且支撑有效
- 操作：在区间底部，且指标超卖（RSI低位）时，**限价挂单**。
- 入场方式：挂在支撑位上方附近。
- 止损设置：**合理止损**。设在区间下沿下方。
- 止盈目标：**区间上沿**。
