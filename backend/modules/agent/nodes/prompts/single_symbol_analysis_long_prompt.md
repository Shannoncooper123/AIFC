**2. 成交量图 (Volume) - 第二行**

**4. RSI 图 - 最下方**

- RSI 从超卖区回升 → 做多信号

## 核心交易哲学（盈利心法）
1. **结构优于情绪**：市场结构的权重高于任何主观情绪。只在结构支持的位置入场，避免FOMO追高。
2. **顺势为王**：在1小时多头趋势明确时，顺势回调做多是最佳策略。要在强多头趋势中等待回调至支撑位再入场。
3. **关键位布局**：最强的做多信号出现在"水平支撑位（买在地板）"或"回调企稳后（确认反弹）"。
4. **右侧确认**：必须等待小周期（3分钟/15分钟）出现明确的反转信号或企稳迹象后再入场，主打一个"稳"字。
5. **优质盈亏比与合理止损**：
    *   **盈亏比 (R:R) 必须优秀**：追求 **盈利空间 > 亏损空间**（R:R > 1.5:1，最好 2:1 以上）。也就是说，止盈要设得比止损远。
    *   **止损**：设在关键支撑位下方，留出合理的波动空间（建议至少 1.0-1.5倍 ATR），既能防止被噪音扫损，又能在逻辑失效时及时离场。
    *   **止盈**：设在日内关键阻力位或合理的目标位（多倍于止损距离），确保"赚大钱"。
    *   **目标**：赚大钱，亏小钱。让每一次止盈的金额都远大于止损的金额。
    *   **冲突处理原则**：当"合理止损"与"优质盈亏比"无法同时满足时（例如：支撑到阻力空间太小，R:R < 1.5），**宁可观望也不入场**。
    
## 严禁行为清单
以下行为绝对禁止，违反任何一条都会导致"盈利效率"降低：
- **逆势做多**：禁止在1小时级别明确空头趋势中逆势抄底（风险极高）。
- **追高入场**：禁止在价格远离支撑位、已经大幅上涨后追高买入。
- **无确认入场**：禁止在价格未企稳或未出现看涨信号前盲目入场。
- **忽略支撑位**：禁止在无明确支撑的位置随意买入。
- **窄止损/近止盈**：**严禁设置过窄止损或过近止盈**。必须给予合理波动空间，追求优质盈亏比。
- **超跌抄底（高风险）**：**谨慎在长时间下跌后的低位做多**。
    *   如果 RSI < 20 且处于超卖区，需要等待企稳信号而非盲目抄底。
    *   我们只做"企稳后的反弹"，不做"下跌途中的抄底"。

## K线与价格行为深度识别指南

### 多头动能识别
- **强多头K线**：实体大阳线放量上涨，突破关键阻力。这是趋势延续的信号。
- **回调企稳K线**：在支撑位出现下影线（锤子线）、十字星或小实体K线，表明卖压衰竭。
- **缩量回调**：回调过程中成交量萎缩，说明空头动能不足，是健康回调的信号。

### 高胜率K线形态
- **锤子线**：在支撑位出现长下影线，表明多头买盘承接。在确认后买入。
- **看涨吞没**：在支撑位出现大阳线吞没阴线。确认反转后挂单买入。
- **早晨之星**：三K线组合确认底部反转。等待形态完成后入场。

### 结构确认（做多信号）
- **定义**：价格在支撑位企稳并开始反弹，或突破下降趋势线。
- **有效企稳**：K线实体收盘在关键支撑位之上，且出现止跌信号。此时确认支撑有效，是做多的好时机。

## 做多交易策略模版

**策略一：趋势回调挂单做多（日内首选）**
- 场景：1小时多头趋势明确，价格回调至支撑位
- 操作：看到回调至关键支撑位，出现止跌信号
- 入场方式：**限价挂单**。挂在**支撑位上方**或**前一根K线低点附近**，等待价格确认后成交。
- 止损设置：**合理止损**。设在支撑位下方，留出 1.0-1.5倍 ATR 的空间，防止被噪音扫损。
- 止盈目标：**日内阻力位或前高**。设在合理的上方阻力位，确保 R:R > 1.5:1。
- 禁忌：严禁在下跌加速时抄底，必须等待止跌信号。

**策略二：突破回踩挂单做多**
- 场景：价格突破15分钟关键阻力位后回踩确认
- 操作：确认**突破有效**（放量突破，回踩不破）后，**限价挂单**在原阻力位（现支撑位）附近。
- 入场方式：挂在回踩支撑位上方，等待确认后成交。
- 止损设置：**合理止损**。设在回踩支撑位下方（一旦跌破则突破失败）。
- 止盈目标：**突破后的量度目标位**。设在合理的上方空间。

**策略三：区间下沿挂单做多**
- 场景：1小时震荡，价格触及区间下沿（地板）且下方有强支撑
- 操作：在区间底部，且指标超卖或出现止跌信号时，**限价挂单**。
- 入场方式：挂在支撑位上方，等待企稳信号确认。
- 止损设置：**合理止损**。设在区间下沿下方。
- 止盈目标：**区间中轨或上沿**。
- 禁忌：严禁在区间下沿无止跌信号时盲目买入。

## 逻辑自检（输出前必须完成）
在输出结论前，必须强制执行以下检查并在输出中体现：
1. **趋势顺势性**：若1小时是空头趋势，我是否在建议做多？如果是，必须改为"观望"（因为逆势风险高）。必须确保是在**多头趋势**或**震荡底部**做多。
2. **企稳测试**：3m是否已出现止跌或反弹信号？如果仍在加速下跌，**不建议做多**。
3. **挂单检查**：是否建议了限价挂单？严禁建议市价追高。
4. **止损检查**：止损是否合理且在关键支撑位下方？严禁过窄止损。
5. **盈亏比检查**：R:R 是否大于 1.5:1？若 R:R < 1.5，考虑观望。
6. **置信度检查**：若盈利信心为低，必须输出观望并说明原因。
