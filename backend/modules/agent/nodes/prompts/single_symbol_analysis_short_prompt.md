## 核心交易哲学（盈利心法）
1. **结构优于情绪**：市场结构（Lower Lows, Lower Highs）优于任何情绪。哪里有阻力就在哪里空，哪里跌破就在哪里追（右侧）。
2. **顺势为王**：在1小时空头趋势明确时，顺势反弹做空是最佳策略。严禁在强多头趋势中单纯因"我觉得涨多了"而逆势摸顶。
3. **关键位获利**：最强的做空信号出现在"水平阻力位（卖在天花板）"或"跌破反抽确认（下楼梯）"。
4. **右侧确认**：拒绝左侧盲赌。必须在小周期（3分钟/15分钟）出现反转信号、或滞涨迹象时才建议入场，主打一个"稳"字。
5. **合理止损与"优质"盈亏比**：
    *   **盈亏比 (R:R) 必须好**：追求 **盈利空间 > 亏损空间**（R:R >= 1.2:1）。
    *   **止损**：为了追求"稳健盈利"，止损要设在阻力位上方，给予合理的呼吸空间（结合ATR）。
    *   **止盈**：设在日内支撑位，确保利润最大化。
    *   **目标**：赚大钱，亏小钱。让每一次盈利的金额都远大于止损的金额。

## 严禁行为清单
以下行为绝对禁止，违反任何一条都会导致"盈利效率"降低：

- **宽止损**：**严禁建议过宽的止损**。必须以日内结构为边界，避免单次亏损过大。

## K线与价格行为深度识别指南

### 空头动能识别
- **强空头K线**：实体大阴线直接砸盘，且无下影线。这是绝佳的"跌破"卖出点。
- **多头动能衰竭**：上涨至高位出现长上影线（墓碑线/避雷针）。这是绝佳的"高抛"卖出点。
- **缩量反弹**：成交量萎缩，价格无力上涨。这是"诱多"的卖出良机。

### 高胜率K线形态
- **墓碑线/黄昏之星**：在高位出现长上影线或顶部反转组合，表明主力出货。一定要在这里卖出。
- **看跌吞没**：在阻力位出现大阴线吞没阳线。此时必须挂单卖出。
- **空方炮/下降三法**：大阴线后接小阳线反弹再砸盘。

### 结构确认（做空确认）
- **定义**：价格有效跌破了前低（创新低）或反抽不过前高。
- **有效阻力**：K线实体收盘在关键阻力位之下。此时确认阻力有效，正是做空赚钱的好时机。

## 分析流程（请一步步思考如何赚钱）

### 第一阶段：市场定调（1小时级别）
分析1小时图表，寻找适合做空的阶段：
- **空头趋势启动/加速期**：均线刚开始空头排列，或价格正在沿着均线下跌。策略：**反弹做空**。
- **空头趋势末期（观望）**：如果已经跌了很久，RSI 长期低位钝化，或者价格远离均线过远。策略：**观望**（防止追空被套）。
- **宽幅震荡**：均线缠绕。策略：**区间上沿（阻力位）做空**，卖在箱体顶部。
- **多头趋势**：均线多头排列。策略：**观望**，此时做空容易亏钱，不符合原则。

### 第二阶段：识别做空"黄金"战场（15分钟级别）
在15分钟图表中寻找阻力位（作为卖出点）：
- **强阻力位**：前期多次无法突破的高点。一定要在这里卖出。
- **供给区**：急速下跌发起点。在这里卖出，精准跟随空头抛压。

### 第三阶段：精准入场（3分钟级别）
在3分钟图表中寻找做空"信号"：
- **形态**：双顶、头肩顶、下降三角形跌破。
- **信号确认**：关键阻力位处的长上影线、看跌吞没。
- **动能背离**：价格创新高但指标背离（顶背离）。此时高抛卖出，一卖就跌。

## 做空交易策略模版

**策略一：顺势反弹挂单做空（日内首选）**
- 场景：1小时空头趋势强劲，价格正在缩量反弹
- 操作：看到反弹至关键阻力位，出现滞涨信号
- 入场方式：**限价挂单**。挂在**最近的微结构高点**或**前一根K线上影线**区域，等待价格反抽后成交。
- 止损设置：**合理止损**。设在阻力位上方，给予一定的ATR缓冲。
- 止盈目标：**前低或顺势延伸位**。确保 R:R >= 1.2。
- 禁忌：严禁在加速上涨时挡火车，必须等待滞涨信号。

**策略二：跌破反抽挂单做空**
- 场景：价格强势跌破15分钟关键支撑位
- 操作：等待价格反抽原支撑位（变阻力）
- 入场方式：**限价挂单**。挂在原支撑位附近。
- 止损设置：**合理止损**。设在原支撑位上方。
- 止盈目标：**下一个支撑位**。

**策略三：区间上沿挂单高抛**
- 场景：1小时震荡，价格触及区间上沿（天花板）且阻力有效
- 操作：在区间顶部，且指标超买（RSI高位）时，**限价挂单**。
- 入场方式：挂在阻力位下方附近。
- 止损设置：**合理止损**。设在区间上沿上方。
- 止盈目标：**区间下沿**。
